O que é React? React é uma biblioteca JavaScript de código aberto para construir interfaces de usuário, especialmente para aplicações de página única. Ele permite que os desenvolvedores criem componentes reutilizáveis que gerenciam seu próprio estado, facilitando a construção de interfaces complexas e interativas. React foi desenvolvido pelo Facebook e é amplamente utilizado na indústria para criar aplicativos web modernos. A categotia delas é SPA (Single Page Application), ou seja, aplicações de página única, onde o conteúdo é carregado dinamicamente sem a necessidade de recarregar a página inteira. React utiliza um DOM virtual para otimizar as atualizações da interface do usuário, tornando as aplicações mais rápidas e responsivas. Arquitetura do React: 1. Componentes: React é baseado em componentes, que são blocos de construção reutilizáveis. Cada componente pode ter seu próprio estado e propriedades, permitindo a criação de interfaces modulares e fáceis de manter. 2. JSX: React utiliza uma sintaxe chamada JSX, que permite escrever código HTML dentro do JavaScript. Isso torna a criação de componentes mais intuitiva e legível. 3. Estado e Props: O estado (state) é usado para armazenar dados que podem mudar ao longo do tempo, enquanto as propriedades (props) são usadas para passar dados de um componente pai para um componente filho. Essa separação ajuda a manter a lógica de negócios e a apresentação da interface organizadas. Pré Requisitos para usar React Para iniciar uma aplicação com React, é necessário ter o Node.js instalado em sua máquina. O Node.js é um ambiente de execução JavaScript que permite rodar código JavaScript fora do navegador. Além disso, é recomendável ter conhecimentos básicos de HTML, CSS e JavaScript. Gerenciador de Pacotes: O npm (Node Package Manager) é o gerenciador de pacotes padrão para o Node.js e é usado para instalar bibliotecas e dependências, incluindo o React. Com o npm, você pode facilmente adicionar o React e outras bibliotecas necessárias ao seu projeto. Aplicações com React Utyilização do Vite: O Vite é uma ferramenta de construção rápida e leve para projetos JavaScript modernos, incluindo aplicações React. Ele oferece uma experiência de desenvolvimento otimizada, com recarregamento instantâneo e suporte a módulos ES. Para criar um projeto React com Vite, basta executar o comando `npm create vite@latest my-react-app --template react` no terminal, onde "my-react-app" é o nome do seu projeto. Isso configurará automaticamente um ambiente de desenvolvimento React com Vite, permitindo que você comece a construir sua aplicação imediatamente. Estrutura base do React Pastas : - node_modules: Contém todas as dependências do projeto, incluindo o React e outras bibliotecas necessárias. - public: Contém arquivos estáticos que serão servidos diretamente, como o arquivo index.html, imagens e outros recursos. - src: Contém o código-fonte da aplicação, incluindo componentes React, arquivos CSS e outros arquivos JavaScript. É onde a maior parte do desenvolvimento acontece. - src/index.jsx: O ponto de entrada da aplicação React, onde o componente raiz é renderizado no DOM. - src/App.jsx: O componente principal da aplicação, onde a lógica e a estrutura da interface do usuário são definidas. Criando componentes - Os componentes ficanm dentro de uma pasta chamada "components" dentro da pasta "src". - Nomeadas em CamelCase, ou seja, a primeira letra de cada palavra é maiúscula. Por exemplo: "MyComponent.jsx". - A utilização da extensão .jsx facilita a formatação para os editores de código, permitindo que eles reconheçam o código JSX e apliquem a formatação adequada. - Dentro do componente precisamos criar e exportar uma função, que é a logica dele. Importação de componentes - Para utilizar um componente em outro arquivo, é necessário importá-lo. A sintaxe de importação é a seguinte: `import ComponentName from './ComponentPath';`, onde "ComponentName" é o nome do componente que você deseja importar e "ComponentPath" é o caminho relativo para o arquivo do componente. - Para inisiar o componente precisamos renderizá-lo dentro do arquivo "src/index.jsx". A sintaxe para renderizar um componente é a seguinte: ``, onde "ComponentName" é o nome do componente que você deseja renderizar.
